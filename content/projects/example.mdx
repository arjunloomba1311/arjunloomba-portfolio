# Example MDX File

This is a sample MDX file created to demonstrate the basic structure and content. MDX allows you to write JSX in your Markdown documents.

## Features

- Write JSX directly in your markdown files.
- Easy to integrate with static site generators.
- Great for documenting components and libraries.
